About this example application

This sample application demonstrates how easy it is to develop stateful web applications using Java EE 6 with portable Seam modules. Just register, login, and book a room to see the future of Java EE in action. Throughout the application you'll see popup links like the ones at the bottom of this page. Click them to see how the application works under the hood.
